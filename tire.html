Tire Maintenance System in Dumper Trucks

Overview

This Tire Maintenance System is built specifically for dumper trucks used in mining and heavy-duty operations. It uses pressure and temperature sensors on each tire, a hydraulic pressure transducer for load detection, and GPS for speed tracking to calculate TKPH (Ton-Kilometer per Hour). Data is sent to a Firebase-linked React Native dashboard for real-time monitoring, helping prevent tire failures, reduce downtime, and improve operational safety. The project was showcased at the Smart India Hackathon 2024.

Core Components

  • Microcontroller (ESP32/ESP8266)
  • Pressure and Temperature Sensors
  • Hydraulic Suspension with Pressure Transducer
  • Firebase for Cloud Data Storage
  • Responsive Native App Dashboard

Key Features

  • Real-time tire pressure and temperature monitoring (TPMS)
  • Live TKPH calculation using sensor and GPS data
  • Tire health report

Outcome

The system predicted tire health, optimized vehicle maintenance, and helped to enhance safety in heavy-duty dumper operations.
